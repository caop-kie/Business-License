# Text Renderer
Generate text images for training deep learning document KIE model.
Support both latin and non-latin text.

# Setup
- Ubuntu 16.04
- python 3.5+

Install dependencies:
```
pip3 install -r requirements.txt
```

# Synthesize Business Licenses
```
python main.py
```
